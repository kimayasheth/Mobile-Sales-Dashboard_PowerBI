# 📱 Mobile Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Mobile Sales Analysis Dashboard** built using **Power BI**. The dashboard analyzes mobile sales performance across different brands, cities, payment methods, and time periods to help stakeholders make data-driven business decisions.

The objective of this project is to identify sales trends, customer purchasing behavior, and regional performance through interactive visualizations and key performance indicators (KPIs).

---

## 🎯 Business Problem

Businesses generate large volumes of sales data but often struggle to quickly identify:

- Top-performing mobile brands and models
- Sales trends over time
- High-revenue cities
- Customer payment preferences
- Overall sales performance

This dashboard provides a centralized view of these insights to support better business decisions.

---

## 🛠 Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel (Dataset)

---

## 📊 Key Performance Indicators (KPIs)

- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Unit Price

---

## 📈 Dashboard Features

- Sales by City (Map)
- Monthly Quantity Trend
- Sales by Mobile Model
- Customer Ratings Analysis
- Payment Method Distribution
- Interactive Slicers
  - Brand
  - Mobile Model
  - Payment Method
  - Day Name

---

## 🧹 Data Preparation

Data was cleaned and transformed using Power Query by:

- Removing duplicate records
- Handling missing values
- Standardizing brand and model names
- Correcting data types
- Formatting date columns

---

## 📌 Key Insights

- Apple generated the highest total sales.
- iPhone SE was the top-selling mobile model.
- UPI was the most preferred payment method.
- Metro cities contributed the highest sales.
- Weekend sales were higher compared to weekdays.

This dashboard can help businesses:
